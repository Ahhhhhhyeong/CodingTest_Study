### 정의 및 의미 차이

- **최대공약수(Greatest Common Divisor)** : 두 개 이상의 수의 **공통된 약수(공약수) 중에서 가장 큰 수** 의미한다. 약수는 어떤 수를 나누어떨어지게 하는 수로, 숫자 안에 포함된 모든 구성 요소를 말한다. 공약수는 항상 최대공약수의 약수라는 특징이 있다.
- **최소공배수(Least Common Multiple)** : 두 개 이상의 수의 **공통된 배수(공배수) 중에서 가장 작은 수**를 의미한다. 배수는 어떤 수에 정수를 곱해 나가는 수이며, 공배수는 항상 최소공배수의 배수가 된다.

### 구하는 방식의 차이 (소인수분해 기준)
- **최대공약수** : 각 수에 **동시에 들어 있는 소인수들을 모두 긁어모아** 구한다.
    - ex)  12(2^2 ×3)와 18(2×3^2)에서 공통으로 들어 있는 2 한 개와 3 한 개를 곱한 6이 최대 공약수
    - 즉, 공통된 부분만 추출하는 개념이다.
- **최소공배수** : 소인수분해를 이용할 때, 각 소인수 중 지수가 가장 큰것들을 모두 곱하여 구한다.

## 유클리드 호제법(Euclidean Algorithm)

내가 기존에 알고 있던 `니은(ㄴ)`을 그려 공통된 수로 계속 나누는 방식은 프로그래밍에서 **유클리드 호제법**으로 구현된다.

### 원리

두 수 a,b가 있을 때, a를 b로 나눈 나머지(r)를 구한다.    
다시 b를 r로 나누는 과정을 반복하다가 나머지가 0이 되었을 때, 나누는 수가 바로 최대공약수이다.


### 코드 로직

#### 최대공약수 구하기 

**재귀 적용 안 한 버전**
```javascript
function gcd(a, b) {
    while(b != 0){
        let r = a % b;
        a = b;
        b = r;
    }
    return a;
}
```
a와 b를 나눈 나머지를 r에 저장하고 b의 값을 a에, r의 값을 b에 넣어주는 것을 b가 0이 될 때까지 반복한다.   
b가 0이 되면, a의 값이 최대최대공약수가 된다.


**삼항연산자, 재귀 활용**

```javascript
const gcd = (a, b) => (a % b === 0 ? b : gcd(b, a % b));
```

- a와 b의 나머지가 0이라면, 그 때 b를 반환한다.
- 위의 조건을 만족하지 못한다면 gcd 함수를 재귀적으로 실행한다.
    - 이 때의 a는 이전 b 값이 되며, b는 이전의 r값이 된다.

#### 최소공배수 구하기

**재귀 적용 안 한 버전**

```javascript
function lcm(a, b){
   return a * b / gcd(a, b); 
}
```

최소공배수를 구하는 공식은 (두 수의 곱 / 최대공약수) 이다.


**삼항연산자, 재귀활용**
```javascript
const lcm = (a, b) => (a * b) / gcd(a, b);
```
