# CodingTest Study

Java와 Node.js(JavaScript)로 알고리즘 문제 풀이와 자료구조를 공부한 기록입니다.

## 학습 현황

| 언어 | 풀이 파일 | 주요 플랫폼 | 학습 범위 |
| --- | ---: | --- | --- |
| Java | 47개 | Baekjoon, 자체 자료구조 정리 | 기초 구현, 자료구조, DP, 그래프 탐색, MST |
| Node.js | 122개 | Baekjoon, Programmers, Codewars, Codetree | 기초 문법, 수학, 정렬, 재귀, DP, 백트래킹, 그래프 탐색, 자료구조 |

## Java

### 공부한 알고리즘 및 자료구조

- 기초 입출력, 조건문, 반복문
- 배열, 문자열, 기본 구현
- 스택과 큐
- 후위 표기식 계산
- 시뮬레이션
- 완전 탐색과 백트래킹
- 동적 계획법(DP)
- BFS, DFS 기반 그래프 탐색
- 우선순위 큐
- 유니온 파인드
- 최소 신장 트리(MST)

### 관련 디렉터리

- `Java/Baekjoon/beginner`: Java 기초 문법과 기본 문제
- `Java/Baekjoon/DP`: 동적 계획법 문제
- `Java/Baekjoon/Problem_*`: 백준 개별 문제 풀이
- `Java/Stack_Queue`: 스택과 큐 개념 정리 및 예제
